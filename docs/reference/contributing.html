Contributing to Winhance

Thank you for your interest in contributing to Winhance! Community feedback is essential for making this tool better for everyone.

üöÄ A Note on Development

Winhance is a solo-developer project. I am building this tool not only as a utility for the community but also as a primary learning experience. I do not have a traditional development background, and this project is my hands-on way to learn and improve my software development skills.

How You Can Help

The most valuable way to contribute is by participating in the GitHub Issues tracker. Your input helps guide the project's direction and ensures stability.

  • Report Bugs: Found something that doesn't work? Detailed bug reports help me track down and fix issues.
  • Feature Requests: Have an idea for a new optimization or feature? I'd love to hear it!
  • Provide Feedback: Share your experience using Winhance and let me know how it can be improved.
  • Support Development: If you find Winhance helpful, please consider making a small donation via Ko-fi. Your support helps keep the project completely free and open source.
