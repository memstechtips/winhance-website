Installation

Winhance can be installed using several methods depending on your preference. All methods are safe and use the official signed binaries.

⚠️ Administrator Privileges Requested

Winhance requires administrator privileges to modify system settings, remove bloatware, and optimize your Windows installation. Please ensure you run the installer as an administrator.

Method 1: PowerShell One-Liner (Recommended)

The fastest way to install Winhance is through a single PowerShell command. This automatically downloads the latest installer and launches it for you.

powershell 
irm "https://get.winhance.net" | iex

Steps:

  1. Right-click the Start button and select Terminal (Admin) or PowerShell (Admin).
  2. Copy and paste the command above into the window.
  3. Press Enter and wait for the installer to download and launch.

Method 2: Manual Download

If you prefer a manual download, you can get the installer directly from our website or GitHub.

  • Official Website: winhance.net (Click the Download button)
  • GitHub Releases: Latest Release

Method 3: Portable Version

During the installation wizard, you have the option to choose a Portable installation. This is ideal if you want to run Winhance from a USB drive or don't want to install it permanently on your system.

ℹ️ Portable Mode

The portable version offers the exact same features as the installable version but keeps all settings in its local folder.

System Requirements

Before installing, ensure your system meets the following requirements:

Operating System

  • Windows 10: 64-bit (x64), version 2004 or later
  • Windows 11: 64-bit (x64), version 23H2 or later

Software

  • .NET 9 Desktop Runtime: Required to run Winhance. The installer will automatically install this if not present.
  • PowerShell 5.1+: Included with Windows 10/11, required for some advanced features.

Other Requirements

  • Administrator privileges: Required for system-level changes.
  • Internet connection: Required for downloading Winhance and installing external apps via WinGet. Core optimization features work offline.
