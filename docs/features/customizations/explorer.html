File Explorer Customization

Enhance the Windows File Explorer with classic features, improved visibility, and powerful context menu additions. Winhance gives you complete control over how File Explorer looks and behaves.

Registry Details

Each setting below includes expandable registry details for power users. Click "Registry Details" to see the exact registry keys and values that are modified.

Context Menu

Customize the right-click context menu with classic features and useful additions.

Use Classic Context Menu explorer-customization-context-menu Windows 11 Only

Use the Windows 10-style right-click menu with all options visible instead of the simplified Windows 11 menu.

Registry Details
PathHKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32
Value(Default)
TypeString
EnabledEmpty string
DisabledKey deleted
DefaultKey absent

Note: Requires Explorer restart.

Add 'Take Ownership' to Context Menu explorer-take-ownership

Adds a right-click option to take ownership of files, folders, and drives with automatic permission elevation.

Registry Details

This setting creates multiple registry keys under:

  • HKCR\*\shell\TakeOwnership
  • HKCR\Directory\shell\TakeOwnership
  • HKCR\Drive\shell\runas

Requires elevation to install. Based on script by Shawn Brink (TenForums).

Add 'Show/Hide Extensions' to Context Menu explorer-context-menu-toggle-extensions

Adds a right-click menu option to quickly toggle file extension visibility in File Explorer (only visible on the Classic Context Menu or Show More Options Menu in Windows 11).

Registry Details

ValuesCommandStateSync, Description, ExplorerCommandHandler, MUIVerb
PathHKCR\Directory\Background\shell\Windows.ShowFileExtensions
Handler{4ac6c205-2853-4bf5-b47c-919a42a48a16}

File Associations

Configure default applications for file types.

Enable Windows Photo Viewer explorer-enable-photo-viewer

Restore the legacy Windows Photo Viewer and set it as the default program for common image file formats (.bmp, .gif, .jpg, .jpeg, .png, .tiff, and more).

Registry Details

This setting associates the following file types with PhotoViewer.FileAssoc.Tiff:

  • .bmp, .cr2, .dib, .gif, .ico, .jfif, .jpe, .jpeg, .jpg, .jxr, .png, .tif, .tiff, .wdp

Registry keys created under:

  • HKCU\SOFTWARE\Classes\.[ext]
  • HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FileExts\.[ext]\OpenWithProgids
