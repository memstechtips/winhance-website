Start Menu Customization

Declutter your workspace by removing unwanted pins and bloatware from the Windows Start Menu.

Key Features

  • Clean Start Menu: A one-click solution to unpin all default apps and third-party bloatware (like LinkedIn ghost icons). When executed, you can also apply recommended start menu settings to disable suggestions and tracking features.
  • Remove Recommendations: Reclaim space by removing the "Recommended" section entirely from the Windows 11 Start Menu.
    Note: Full removal requires Windows 11 Pro or higher. On Home editions, the section will be disabled but still visible.
  • Disable Bing Search: Prevent the Start Menu from showing online search results from Bing, keeping your searches local and private.
⚠️ Setting Persistence

Settings found in the Customizations view are not persistent. Winhance applies these changes once, but it does not use background tasks to re-apply them if they are changed later by Windows or by you manually.

Layout

Clean Start Menu start-menu-clean-10 Windows 10 Only

Removes all pinned tiles and applies a clean Start Menu layout for all users on this computer. This action requires confirmation.

Type: Action button

How It Works

Windows 10 uses an XML-based layout system. Winhance creates a clean layout file and applies it through Windows policies:

  • Current User: The layout is applied immediately by temporarily locking the Start Menu layout via registry policy, restarting the Start Menu process, then unlocking it so you can customize again.
  • Other Existing Users: Scheduled tasks are created under Winhance\CleanStartMenu_{username} that run once when each user next logs in. The task applies the layout and then self-deletes.
  • New Users: The layout file is saved to the Default user profile, so any new user accounts automatically get the clean layout.
Technical Details

Registry PathHKCU\SOFTWARE\Policies\Microsoft\Windows\Explorer
Registry ValuesLockedStartLayout (DWORD), StartLayoutFile (String)
Process RestartedStartMenuExperienceHost
Layout File Contents 
<?xml version="1.0" encoding="utf-8"?>
<LayoutModificationTemplate
    xmlns:defaultlayout="http://schemas.microsoft.com/Start/2014/FullDefaultLayout"
    xmlns:start="http://schemas.microsoft.com/Start/2014/StartLayout"
    Version="1"
    xmlns:taskbar="http://schemas.microsoft.com/Start/2014/TaskbarLayout"
    xmlns="http://schemas.microsoft.com/Start/2014/LayoutModification">
    <LayoutOptions StartTileGroupCellWidth="6" />
    <DefaultLayoutOverride>
        <StartLayoutCollection>
            <defaultlayout:StartLayout GroupCellWidth="6" />
        </StartLayoutCollection>
    </DefaultLayoutOverride>
</LayoutModificationTemplate>

This XML defines an empty Start layout with no pinned tiles.

Clean Start Menu start-menu-clean-11 Windows 11 Only

Removes all pinned apps and applies a clean Start Menu layout for all users on this computer. This action requires confirmation.

Type: Action button

How It Works

Windows 11 uses a binary file system for Start Menu pins and a JSON-based policy for configuration. Winhance cleans the Start Menu through multiple approaches:

  • Current User: Deletes the start.bin and start2.bin files from the current user's LocalState folder, then restarts the Start Menu process.
  • Other Existing Users: Deletes the same binary files from all other user profiles on the system (requires admin privileges).
  • New Users: A system-wide policy is applied via ConfigureStartPins that configures an empty pinned list, ensuring new users get a clean Start Menu.
Technical Details
Binary Files Deleted%LocalAppData%\Packages\Microsoft.Windows.StartMenuExperienceHost_cw5n1h2txyewy\LocalState\start.bin
%LocalAppData%\Packages\Microsoft.Windows.StartMenuExperienceHost_cw5n1h2txyewy\LocalState\start2.bin
Policy Registry PathHKLM\SOFTWARE\Microsoft\PolicyManager\current\device\Start
Policy ValueConfigureStartPins = {"pinnedList":[]}
Process RestartedStartMenuExperienceHost

Note: The start.bin and start2.bin files store the user's pinned apps. Deleting them resets the pins to the system default (which is empty due to the policy).
