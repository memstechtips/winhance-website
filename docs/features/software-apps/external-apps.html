External Software (WinGet Integration)

Winhance integrates with Microsoft WinGet to provide a seamless way to install over 100 popular third-party applications. This includes browsers, development tools, gaming platforms, and more.

Key Features

  • No Wait: Installation is performed silently in the background.
  • Official Sources: Apps are downloaded directly from the developers' official repositories.
  • Batch Install: Select multiple apps and install them all with a single click.

Winhance bundles WinGet with the application to ensure it is always available. If a system-installed WinGet is found (installed via Microsoft Store), it is used for install and uninstall operations. The bundled copy is used as a fallback on systems without WinGet (e.g., LTSC editions).

‚ÑπÔ∏è Chocolatey Fallback

If a WinGet install or uninstall fails for certain apps, Winhance can fall back to Chocolatey. You will be prompted for consent before Chocolatey is used. Chocolatey is also used for installation status detection as a secondary source.

Winhance scans your system using WinGet to determine if an app is already present. This allows you to quickly see what you have and what you might want to install or re-install.

üí° Pro Tip: Website Links

Next to each application name, you'll find a small external link icon. Clicking this will take you directly to the developer's official website so you can learn more about the software before installing.

Managing Software

For more advanced software management, we highly recommend using UniGetUI (formerly WinGetUI). It's a powerful graphical interface for various package managers including WinGet, Scoop, and Chocolatey. You can install UniGetUI directly through Winhance in the Utility category.

‚ûï Requesting New Software

If you have an application you'd like to see added to Winhance, please visit our GitHub repository and create a new issue marked as a Feature Request. We prioritize adding apps that are available via WinGet and are widely used by the community.
