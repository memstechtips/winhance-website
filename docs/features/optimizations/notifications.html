Notifications

Reduce distractions and take control of Windows notifications. Winhance lets you customize which notifications you receive, silence unwanted alerts, and remove promotional content.

Registry Details

Each setting below includes expandable registry details for power users. Click "Registry Details" to see the exact registry keys and values that are modified.

Notification Controls

Core settings for Windows notification behavior.

Show Notifications windows-pushnotifications

Get notifications from apps and other senders in Windows. Disabling this is the master switch for all toast notifications.

Registry Details
PathHKCU\Software\Microsoft\Windows\CurrentVersion\PushNotifications
ValueToastEnabled
TypeDWORD
Enabled1
Disabled0
Default1 (Absence = Enabled)

Note: Changing this setting restarts the WpnUserService.

Allow notifications to play sounds notifications-sound

Play audio alerts when notifications arrive from apps and system senders. Requires "Show Notifications" to be enabled.

Registry Details
PathHKCU\Software\Microsoft\Windows\CurrentVersion\Notifications\Settings
ValueNOC_GLOBAL_SETTING_ALLOW_NOTIFICATION_SOUND
TypeDWORD
Enabled1
Disabled0
Default1 (Absence = Enabled)
Show notifications on the lock screen notifications-toast-above-lock

Display toast notifications on the lock screen when your device is locked. Requires Lock Screen to be enabled.

Registry Details
PathHKCU\Software\Microsoft\Windows\CurrentVersion\Notifications\Settings
ValueNOC_GLOBAL_SETTING_ALLOW_TOASTS_ABOVE_LOCK
TypeDWORD
Enabled1
Disabled0
Default1 (Absence = Enabled)
PathHKCU\Software\Microsoft\Windows\CurrentVersion\PushNotifications
ValueLockScreenToastEnabled
TypeDWORD
Enabled1
Disabled0
Default1 (Absence = Enabled)

Additional Settings

Some settings in this section require the "Ads, Suggestions and Promotional Content" setting in Privacy & Security to be set to "Custom" before they can be modified.
