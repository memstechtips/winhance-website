Power Management & Plans

Winhance provides a comprehensive power management interface, allowing you to select from system-default plans or apply specialized optimizations.

Power Plan Selection

The Power Plan ComboBox in Winhance is dynamic and intelligent:

  • Custom Plan Detection: Winhance automatically detects any custom power plans already present on your system and adds them to the selection list.
  • Default Windows Plans: Standard plans such as Balanced, High Performance, and Power Saver are always available.
  • Ultimate Performance: Winhance provides a one-click option to enable and activate the hidden Windows "Ultimate Performance" plan, which is designed for high-end systems to minimize micro-latencies.
  • Winhance Power Plan: This is a custom preset included with Winhance. It is based on the Ultimate Performance plan but further tuned for the best possible responsiveness.
  • Delete Plans: You can remove unwanted power plans from your system by clicking the Trash Can icon next to the plan name in the dropdown. Note that default Windows schemes can be easily restored if deleted.
‚ÑπÔ∏è Advanced Power Settings

Winhance unlocks hidden "Advanced Power Settings" that are normally invisible in the Windows Control Panel. This includes settings like Processor Boost Mode, Core Parking, and Energy Performance Preference.

Warning: These settings are locked by default for safety. Unlocking and modifying them (especially numeric values) can cause system instability or increased heat generation if set incorrectly. Winhance will show a warning dialog before allowing you to unlock these controls.

üíª Laptop vs Desktop View

Winhance intelligently detects if your device has a battery. On a Desktop, you will see a single set of controls (AC Power). On a Laptop, you will see two sets of controls: one for Plugged In (AC) and one for On Battery (DC), allowing for granular control over power consumption.

‚ö†Ô∏è Setting Persistence

Settings found in the Optimizations and Customizations views are not persistent. Winhance applies these changes once when you click "Apply", but it does not use background tasks to re-apply them if they are changed later by Windows or by you manually.

This is a deliberate design choice to respect user choice. For example, if you set Dark Mode via Winhance but later decide to switch back to Light Mode using the official Windows Settings app, Winhance will not revert your change on the next reboot.

Configuration Export & Power Plans

When you export your Winhance configuration to a .winhance file:

  • If a custom power plan is active, the config file will attempt to re-apply that plan when imported on another system.
  • Note: While the plan itself will be selected, some of the advanced power settings (like specific sub-setting values) might not be identical if the target system has different hardware capabilities or if those settings weren't explicitly defined in the custom plan.
